# 图卡库 V2 需求说明文档

## 1. 概述
图卡库 V2 旨在为特殊教育工作者提供一个智能化、高效的视觉支持资产管理平台。平台通过整合 AI 生成、智能语义检索、以及灵活的层级分类管理，帮助教师快速沉淀、查找和归类教学图卡。

---

## 2. 智能搜索功能

图卡库提供智能搜索工作台，大幅提升教师检索教具与图卡的效率。

### 2.1 进入与退出机制
- **进入**：点击进入，页面可切换并聚焦进入智能搜索模式。
- **退出**：点击退出，可平滑退出智能搜索模式，恢复到图卡库标准视图。

### 2.2 检索机制优先级
- **优先语义搜索**：用户输入自然语言，后台通过 Embedding 向量进行语义图卡检索。
- **降级机制**：当语义检索未能搜到足够或相关度高的内容时，自动启动传统特征的**关键词匹配**。

### 2.3 辅助搜索：教案与活动关联搜索
用户可通过弹窗单选具体的【教案】或【活动】，系统会自动提取其中的内容属性转换为结构化的检索指令。只能同时添加一个教案或活动。

#### 2.3.1 教案搜索 Prompt 设定
选定教案后，系统后台自动调用大语言模型解析内容并进行查询：
```text
# Role
你是一名资深的特殊教育专家和儿童认知图卡语义解析引擎。你非常了解特殊儿童在各类教学活动中需要的视觉支持，并擅长将枯燥的教案属性转化为生动的画面描述。

# Task
你的任务是根据用户提供的【教案详情】，生成用于“向量数据库”检索的【语义描述语句】，以及用于关键词搜索的【关键词】。

# Input Format
- 教案名称：[教案名称]
- 教案活动：[活动列表（活动名称-活动步骤-训练目的）]
- 课程纲要：[课程纲要]

# Guidelines
1. 【语义检索语句 (Semantic Query)】：用自然语言描述这个教案中可能出现的所有高频视觉元素、核心动作、人物互动状态和所需物品。这句话是给机器（Embedding模型）看的，语料越丰富、画面感越强越好。
2. 【关键词 (Key words)】：提炼出3-5个最核心的名词或动词短语。用于关键词搜索匹配。

# Output Format
请严格以 JSON 格式输出，不要包含任何额外的解释文本：
{
  "semantic_query": "一段富有画面感、包含多个元素的描述性文本，用于生成向量",
  "key_words": ["关键词1", "关键词2", "关键词3"]
}
```

#### 2.3.2 活动搜索 Prompt 设定


### 2.4 搜索结果交互
- 在“我的图卡”区切换显示为 **“搜索结果”** 视图。
- 显示的所有关联图卡均支持**直接鼠标拖拽**，允许分类存放。

## 3. 分类管理功能（原文件夹管理）

### 3.1 视图与交互
- **术语变更**：“文件夹”统一更名为“**分类**”。
- **展开/折叠**：分类大区允许自主展开或收缩。
- **进入与退出**：
  - **进入**：点击对应分类卡片即可进入分类详情页面，顶部显示面包屑导航。
  - **退出**：通过面包屑导航返回。

### 3.2 系统内置分类
- **分层管理**：包括基础认知层、核心功能层、社会化层等。
- **校区图卡**：特殊的系统内置分类，**动态显示全校区所有用户生成的图卡集合**。
  - **只读属性**：不支持手动将图卡拖入或移入，系统会自动同步全量数据。
  - **无权修改**：无法重命名或删除此分类。
- **保护机制**：系统内置的分层分类严禁被重命名和删除。

### 3.3 用户私有分类
- **隐私保护**：用户自建的分类**仅创建者本人可见**，其他用户无法查看。
- **自主管理**：支持自由重命名、删除及多级嵌套。

---

## 4. 我的图卡（素材全图库）

“我的图卡”是用户个人资产的核心展示区，取代了原有的“未分类”逻辑。

- **全量持久化**：
  - “我的图卡”展示用户上传及 AI 生成的所有个人图卡。
  - **不随分类消失**：图卡被拖入或移动到某个分类后，**依然会保留在“我的图卡”列表中**，作为素材全库供随时调阅。
- **删除逻辑（双重确认）**：
  - **在分类内删除**：仅将图卡从该特定分类中移出，不会影响其在“我的图卡”全库中的存在。
  - **在“我的图卡”中删除**：彻底从系统中永久删除该图卡。
- **交互增强**：
  - 支持折叠/展开“我的图卡”区域。
  - 支持右键点击图卡进行“移动到...”操作。
- **批量操作**：支持全选、批量打印、批量下载、批量删除。

---

## 5. AI 生成图卡工作台

### 5.1 生成与管理
- **任务私有化**：任务列表仅展示当前登录用户创建的生成记录。
- **状态筛选**：支持按“全部状态、生成中、待处理、已完成、生成失败、部分成功、已取消”进行任务过滤。

### 5.2 核心操作项
- **复用功能**：一键回填历史生成参数，支持二次微调。
- **新增打印项**：
  - **单张打印**：每张已生成的图卡均支持独立打印。
  - **批量打印**：支持针对整个生成任务的所有图卡进行一键批量打印。

---

## 6. UI/UX 体验优化

- **去分页化**：移除图卡库的分页器，采用更流畅的滚动加载体验。
- **边缘避碰**：右键菜单及悬浮操作栏支持智能位置调整，针对最后一行图卡自动反转弹出方向，防止内容被屏幕边缘遮挡。
- **菜单精简**：移除非必要的“从分类移除”等冗余右键选项，操作路径更聚焦。
