## 2. 页面详细需求

### 2.1 图卡库 - AI生成图卡

### 2.1.1 功能描述

用户通过AI描述词快速生成教学图卡，支持多种风格和尺寸选择，一次生成4张。生成后数据展示在图卡库-图卡页面，存入图卡库-全部页面。

### 2.1.2 用户故事

> 作为一名特教老师,我希望通过简单的描述就能快速生成适合儿童认知的图卡,这样可以节省我大量的制作时间。
> 

### 2.1.3 功能需求

| 功能点 | 详细描述 | 优先级 | 提示词 |
| --- | --- | --- | --- |
| 任务名称输入 | 支持用户输入图卡生成任务的名称 | P0 |  |
| 提示词输入 | 支持用户输入图卡描述提示词 | P0 | 参考图中的人物形象，生成一张儿童认知图卡，{用户输入}。 |
| 提示词优化 | 提供AI优化提示词功能，帮助用户生成更好的描述 | P1 | 我的提示词：{用户输入}，我想要生成一张儿童认知图卡，请优化我的提示词。 |
| 风格选择 | 支持多种风格选择：卡通、写实、简笔画、3D、自定义上传参考图 | P0 |  |
| 尺寸选择 | 支持多种尺寸比例：1:1、4:3、3:4、16:9、9:16 | P0 |  |
| 生成按钮 | 点击后开始生成图卡，并创建生成任务 | P0 |  |
| 成功提示 | 生成成功后显示提示，引导用户前往图卡页面查看 | P1 |  |

### 2.1.4 界面设计

- 采用卡片式布局，清晰的表单字段
- 风格和尺寸选择采用可视化的图标和标签
- 生成按钮有加载状态反馈
- 成功提示采用Toast通知形式

### 2.2 图卡库 - 全部

### 2.2.1 功能描述

展示所有图卡资源，支持筛选、搜索、批量操作等功能。

### 2.2.2 用户故事

> 作为一名特教老师,我希望能在一个页面看到所有的图卡资源,并能快速筛选和查找我需要的内容,这样可以提高我的工作效率。
> 

### 2.2.3 功能需求

| 功能点 | 详细描述 | 优先级 |
| --- | --- | --- |
| 图卡列表展示 | 以卡片布局展示所有图卡，包含缩略图、标题、标签等信息 | P0 |
| 搜索功能 | 支持按图卡名称进行搜索 | P0 |
| 筛选功能 | 支持按创建者、图卡类型、发布状态进行筛选 | P1 |
| 分页功能 | 支持图卡分页展示 | P1 |
| 批量选择 | 支持多选图卡进行批量操作 | P1 |
| 批量删除 | 支持批量删除选中的图卡 | P1 |
| 批量下载 | 支持批量下载选中的图卡 | P1 |
| 单个图卡操作 | 支持单个图卡的重命名、发布/取消发布、删除 | P0 |
| 打印 | 维持现有功能 | P1 |

### 2.2.4 界面设计

- 顶部工具栏：搜索框、筛选器、批量操作按钮
- 主内容区：卡片样式展示图卡
- 每个图卡卡片：缩略图、名称、标签、创建者、操作菜单
- 分页组件位于底部

### 2.3.1 功能描述

以任务列表形式展示使用AI生成图卡功能的结果。

### 2.3.2 用户故事

| --- | --- | --- |

### 2.4 图卡库 - 词卡

### 2.4.1 功能描述

专门展示词卡类型的资源，展示活动生成的词卡。

### 2.4.2 用户故事

> 作为一名特教老师,我希望能在词卡页面看到从活动对话中生成的词卡,这样我可以统一管理所有词卡资源。
> 

### 2.4.3 功能需求

| 功能点 | 详细描述 | 优先级 |
| --- | --- | --- |
| 词卡筛选 | 默认只显示type为'word'的词卡 | P0 |
| 活动词卡展示 | 展示从活动对话中生成的词卡，标记来源活动 | P0 |
| 词性标签 | 词卡显示词性标签（名词、动词、形容词等），不同词性不同颜色 | P0 |
| 活动列表 | 可展开展示活动列表，查看活动详情和对话 | P1 |
| 活动对话展示 | 展开活动后显示活动对话内容，包含对话和词卡图片 | P1 |
| 活动打印 | 支持单个或批量打印活动词卡 | P1 |

### 2.4.4 数据同步

- 活动详情页面生成的对话和词卡自动同步到词卡页面
- 活动详情页面后续修改内容自动同步到词卡页面

---

### 2.5 活动详情 - 活动对话 - 一键生成对话

### 2.5.1 功能描述

在活动详情页面中，支持一键生成教学对话，并对对话进行编辑和词性解析。

### 2.5.2 用户故事

> 作为一名特教老师,我希望能快速生成适合教学的对话内容,这样可以节省我设计对话的时间。
> 

### 2.5.3 功能需求

| 功能点 | 详细描述 | 优先级 |
| --- | --- | --- |
| 一键生成对话 | 点击按钮后生成预设的教学对话模板 | P0 |
| 对话编辑 | 支持编辑生成的对话内容，修改老师和学生的对话 | P0 |
| 对话展示 | 区分老师和学生的对话，用不同样式展示 | P0 |
| 完成编辑 | 编辑完成后进入词性解析阶段 | P0 |
| 词性解析 | 自动将对话拆解为单词，并标注词性（名词、动词、形容词、代词、其他） | P0 |
| 词性颜色区分 | 不同词性用不同颜色的标签展示 | P0 |
| 数据同步 | 保存的数据自动同步到图卡库的词卡页面 | P0 |

### 2.5.4 流程设计

1. 初始状态：显示"一键生成对话"按钮
2. 点击生成：显示可编辑的对话内容
3. 编辑对话：用户可以修改对话文本
4. 点击完成：显示解析中状态
5. 解析完成：显示词性标注后的对话

### 2.6 活动详情 - 活动对话 - 一键生成词卡

### 2.6.1 功能描述

在词性解析完成后，支持为对话中的单词一键生成词卡图片。

### 2.6.2 用户故事

> 作为一名特教老师,我希望能为对话中的关键词生成配图,这样可以让教学内容更加生动形象。
> 

### 2.6.3 功能需求

| 功能点 | 详细描述 | 优先级 | 提示词 |  |
| --- | --- | --- | --- | --- |
| 生成词卡入口 | 在词性解析完成后显示"一键生成词卡"按钮 | P0 |  |  |
| 风格选择 | 支持选择生成风格：卡通、写实、简笔画、3D、自定义上传参考图 | P0 | 参考图中的人物形象，生成一张儿童认知词卡，整体纯白背景，上方以圆角图片表达“宝宝”这个词方便儿童理解，下方三行文字第一行是词的中文“宝宝”，第二行是词的英文“baby”，第三行用较小的彩色标签显示词的词性“名词”。 | 引号中的内容为参数：{词}、{词的中文}、{词的英文}、{词性} |
| 尺寸选择 | 支持选择生成尺寸：1:1、4:3、3:4、16:9、9:16 | P0 |  |  |
| 开始生成 | 点击后开始为每个单词生成图片 | P0 |  |  |
| 生成状态 | 显示每个单词的生成状态（加载中/完成） | P0 |  |  |
| 图片展示 | 生成完成后在单词上方展示图片 | P0 |  |  |
| 重新生成 | 支持单个单词重新生成图片 | P1 |  |  |
| 数据保存 | 生成完成后自动保存更新的对话和词卡数据 | P0 |  |  |
| 数据同步 | 保存的数据自动同步到图卡库的词卡页面 | P0 |  |  |

### 2.6.4 流程设计

1. 词性解析完成后，显示"一键生成词卡"按钮
2. 点击按钮，弹出风格和尺寸选择弹窗
3. 确认选择后，开始生成图片
